# Metrix2D Unity3D Game
Full gameplay and highscores version

## Controls
+ A/D or Left/Right == Move block left/right
+ S or Down == Increase down speed of block
+ F5 == Highscores table window
+ F8 == Settings window
+ Backspace == Reset game
+ ESC == Quit game
